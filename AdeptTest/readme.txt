* Для сборки создать build-каталог, использовать "cmake "версия visual studio ../AdeptTest""

* Необходим Firebird 3.0 

* При запуске ввести имя файла "test.txt" и имя БД "%.fdb"

* Результаты работы будут сохранены в файлы "groupBy%Alg" - группировка алгоритмом, "groupBy%Bd" - группировка с помощью запросов к БД.
